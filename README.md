# Aggregation: Async Endpoint
Generate REST endpoints with strict-schema enforcement (Serde) for asynchronously loading data into a mytiki.com data lake. Writes are serialized as [Apache Avro](https://avro.apache.org) files and saved to the staging environment for loading.
